# AI-Powered Marketing Analytics & Customer Intelligence System

---

# Project Overview

This project is an end-to-end AI-powered marketing analytics system developed using Python and Machine Learning techniques.

The main objective of this project is to analyze customer purchasing behavior, predict customer churn, recommend products, and forecast future sales trends using retail transaction data.

---

# Project Features

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- RFM Customer Segmentation
- Customer Lifetime Value (CLV)
- Customer Clustering
- Customer Churn Prediction
- Product Recommendation System
- Sales Forecasting

---

# Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- MLxtend
- Jupyter Notebook

# Dataset Information

Dataset Used:
Online Retail Dataset

The dataset contains:

## 1. K-Means Clustering
Used for customer segmentation.

## 2. Random Forest Classifier
